INTRODUCTION TO HTML

What I've learnt in the introduction to HTML is the use of tags, elements and attributes.An HTML tags must be within two angle brackets( < and >). The placement of tags, having pairs(opening and closing tags) to form an element and we have some tags that do not have a closing tag like image tag, break tag, hr tag, etc. The tags are used to design the text placed in between them.

Tags and their functions

  • body - accepts all tags to be display on a webpage
  • p - Text paragraph
  • h1 - h6 - Text header with progressive decrease in the font size
  • a - Text link
  • b - Bold text
  • u - Underline Text
  • br - Line break
  • i - Text italics
  • img - accepts a source value to display images
  • hr - Horizontal rule
  • em - Text Emphasis
  • mark - mark out text in content
  • del - line through a text to signify deleted text
  • ins - Text insert
  • sup - Text superscript like in date(10th)
  • sub - Text subscript like in base of number(102)
  • input - accepts text,url,file, and dropdown list for form
  • button -
  • head - contains the information about the webpage.
  • title - accept the title of the page
  • table - used to display data in rows and columns
